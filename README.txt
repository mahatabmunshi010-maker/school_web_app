Simple offline School Management App
Files:
- index.html (home)
- students.html
- marks.html
- admissions.html
- certificates.html
- settings.html
- styles.css
- app.js

How to use:
1. Unzip and upload to GitHub Pages OR open index.html locally in a modern browser.
2. Admin login is demo: username=admin password=1234 (prompt-based).
3. Use Admissions to add students, then view from Students. Enter marks on Marks page.
4. Print marksheet/certificate using Print dialog (Save as PDF).

Notes:
- This is a static client-side app storing data in localStorage.
- For production (multi-user, server), migrate to real backend (Flask/Django/Node + DB).
